Common Questions

How do I install the plugin?

Go to WordPress Admin → Plugins → Add New → Upload Plugin. Select the ZIP file and activate.

Where do I add my wallet addresses?

Navigate to WooCommerce → Settings → Payments → WpCrypto → Manage. Add your wallet addresses for each cryptocurrency.

How do I get API keys for Pro version?

Visit Etherscan.io, BscScan.com, and TronGrid.io to create free accounts and generate API keys. See our documentation for detailed steps.

Can I test the plugin before buying?

Contact our sales team for a demo or trial period. We'll help you evaluate if WpCrypto is right for your store.
