Complete guide to installing and configuring WpCrypto

Installation

Requirements

  • WordPress 5.0 or higher
  • WooCommerce 3.0 or higher
  • PHP 7.2 or higher
  • cURL enabled on your server

Installation Steps

  1. Download the WpCrypto plugin ZIP file
  2. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New
  3. Click "Upload Plugin" and select the ZIP file
  4. Click "Install Now" and then "Activate Plugin"
  5. Navigate to WooCommerce ‚Üí Settings ‚Üí Payments
  6. Enable "WpCrypto" and click "Manage"

Configuration

Basic Settings

After activating the plugin, configure the following:

  • Enable/Disable: Turn the payment gateway on or off
  • Title: Payment method name shown to customers (e.g., "Pay with Crypto")
  • Description: Message displayed during checkout

Wallet Configuration

Add your cryptocurrency wallet addresses:

  • Bitcoin (BTC): Your BTC wallet address
  • Ethereum (ETH): Your ETH wallet address (0x...)
  • USDT ERC20: Your Ethereum address for USDT
  • USDT TRC20: Your Tron address for USDT
  • BNB: Your Binance Smart Chain address

Pro Version Features

Automatic Verification Setup

The Pro version requires API keys for automatic blockchain verification:

Etherscan API (for ETH & ERC20 tokens)

  1. Visit etherscan.io/apis
  2. Create a free account
  3. Generate an API key
  4. Paste it in WpCrypto settings

BscScan API (for BNB & BEP20 tokens)

  1. Visit bscscan.com/apis
  2. Create account and get API key
  3. Enter in plugin settings

TronGrid API (for USDT TRC20)

  1. Visit trongrid.io
  2. Register for free API access
  3. Configure in WpCrypto

How Customers Use It

Checkout Process

  1. Customer adds products to cart and proceeds to checkout
  2. They select "Pay with Crypto" as payment method
  3. Choose their preferred cryptocurrency (BTC, ETH, USDT, etc.)
  4. Order page displays:
    • Your wallet address
    • Exact crypto amount to send
    • QR code for mobile wallets
    • Copy button for easy address copying
  5. Customer sends payment from their wallet
  6. Manual Version: Customer clicks "I Paid" button, you verify manually
    Pro Version: System auto-detects payment and completes order

Troubleshooting

Common Issues

Plugin not showing at checkout

  • Ensure plugin is activated
  • Check WooCommerce ‚Üí Settings ‚Üí Payments
  • Enable WpCrypto payment method

Auto-verification not working (Pro)

  • Verify API keys are correctly entered
  • Check API key permissions
  • Ensure cURL is enabled on server
  • System will fall back to manual mode if APIs fail

Wrong crypto amount displayed

  • Plugin uses real-time exchange rates
  • Rates update every few minutes
  • Minor variations are normal
